Explanation

If the size is NOT set as Square or Absolute then the width entered will be used to resize the longest side and the height ignored.

If set to Square then

  1. the image will be resized so that the longest side is the same length as the width
  2. a thumbnail will be created from the center of the resized image
  3. the height entry will be ignored

If set to Absolute

  1. the width and height of the image will be checked and whichever is the smaller will be used to resize the image
  2. a thumbnail will be created from the center of the resized image using the dimensions entered
